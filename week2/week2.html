Bootstrap

Resource: Bootstrap - Bootstrap is a CSS/JavaScript (we’ll learn JavaScript later) framework (pre-written code/instructions, in our case, pre-wrritten CSS). Basically, Twitter (who created Bootstrap) created a framework that makes developing mobile-first web applications easier.

View the BootStrap CSS file here. As a rule of thumb, designers do not modify bootstrap.css. Any additional CSS should be added to a new file, say custom.css.
